# AVL
- 平衡二叉树：每个节点存balance factor = {-1， 0， 1}
- 节点需要存储额外信息，调整次数频繁
# 红黑树
红黑树是一种近似平衡二叉树，确保任何一个节点的左右子树高度差小于两倍
- 不能有相邻的两个红色节点
- 从任一节点到其每个叶子所有路径都包含相同数目黑色节点

# 位运算
- 判断奇偶：(x&1) == 1, (x&1) == 0
- x >> 1 --> x / 2
- x = x&(x - 1) 清零最低位的1（超级常用！！！！！！！！！！！！！！！！）
- x&-x --> 得到最低位的1
- x&~x = 0
